Project Overview

The HMI Dashboard Design project focused on creating a fully custom human-machine interface for a high-throughput manufacturing production line. The interface was engineered to maximize operator efficiency through intuitive layouts, color-coded alarm states, and real-time process feedback.

Developed on the Allen-Bradley FactoryTalk View platform and integrated with a CNC-controlled production environment, the dashboard provides operators with complete situational awareness — from individual machine states to plant-wide performance metrics — all within a single, unified interface.

/

Key Features

  • Multi-screen HMI layout with dynamic process flow diagrams
  • Real-time alarm management with priority queuing and acknowledgment logging
  • Operator-level access control with role-based permissions
  • Integrated production counters, OEE metrics, and shift reporting
  • Custom trend displays for temperature, pressure, and speed parameters
  • One-click machine start/stop with safety interlock visualization
  • Mobile-responsive companion view for supervisor oversight
